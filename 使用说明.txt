【程序所需运行环境】

支持  Apache/IIS/Nginx  + MySQL

PHP版本     5.1.X或更高

MySQL版本  4.0X或更高  需要打开 PDO MySQL扩展模块

PDO MySQL扩展模块打开方法

修改本地php.ini配置，以Windows下的php为例，
找到：
;extension=php_pdo_mysql.dll ;
;extension=php_pdo.dll;
去除前面的分号“;”重启Apache/IIS/Nginx 服务即可。



安装说明：

安装前请先确认您的空间（服务器）达到程序所需运行环境要求。如果您购买的是虚拟主机（空间）无须自己搭建这样的运行环境，只需购买空间时选择购买这样的运行环境即可。

1.在 http://open.114la.com/ 下载自己想使用的 114啦网址导航版本,我们提供了二种不同的编码。包括 GBK 简体中文版、UTF-8中文版，请根据您空间所支持的编码，选择相应版本的程序下载。

2.下载程序完成后，解压把upload目录下的所有文件上传至空间根目录下，推荐使用FlashFXP工具通过ftp传到服务器上，工具下载地址：http://www.xiazaiba.com/html/1814.html

Linux 或 Freebsd 服务器下安装，使用ftp工具中的二进制模式上传。

3. 通过浏览器访问 http://您的域名/安装目录/install/， 根据提示填写 mysql 配置信息、管理员账号信息

4. 程序将自动导入114啦数据库，并生成您刚才的配置信息

5. 若该目录有修改权限，将自动删除install 目录，否则请您自行删除 install 目录

6. 进入管理后台全局设置>>>站点设置>>>设置站点的相关资料>>>然后转到>>>页面生成>>>生成全站。

7. 完成安装！


特别注意：

建议将“后台管理系统”的目录名由默认的 admin 修改为其他目录名。

直接修改根目录下的 admin 目录名为新的目录名.
